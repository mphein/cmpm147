Experiment 4b - Massive Landscape

Description


Infinity

I used infinity in my world to represent a never ending landscape, with oceans, grasslands, mountains, and desert. Also certain seeds generate infinite maps which look like compact atoms in a solid.

Variation in Space

As you can see there are different sizes of terrain and colors. There are also different patterns which occur by seed. Some are more abstract.

Variation in Key

Different keys will show either a terrain with smooth noise transitions or completely random colors at each tile for a wonderfully colorful experience.

Signs of life

There is a sparkle mode that randomizes the color values based on perlin noise, also there is a wiggle on each tile to simulate movement or time passing.

Technical

Using the user's world key I was able to create a world seed from the string which would generate the same perlin noise and random values to preserve values for each key and seed. For rainbow road I was just getting started with generating this infinite landscape. I started by making a roadWidth which would draw a colored tile infinitely. From there I wrote a function to color in each tile using colors from the rainbow. Using perlin noise I was able to achieve different effects such as vertical road lines, horizontal road lines, and random shapes along the road. I also added a sparkle function which will further randomize the color of each tile between a set value which is able to distinguish that color. I also was able to add an offset to the road center such that it appears to be curivng and turning at random points. For the massive landscapes I was able to zoom out by making the tile sizes smaller. I also added some random displacement on the tiles so they appear to be moving or shifting constantly in and out of place. I had the most fun making the interesting tiles (Experiment 4c) and I played around with using random values to shift the tile size which can be controlled by the user. This allows for massive landscapes with height and depth while also creating abstract tile art and patterns.

Reflection

I learned quite a bit from this assignment and grew in creativity along the way. I started with a concrete image and moved more to abstraction, using noise to help me and the user create abstract art. I definitely was very lost with what to do in the beginning, but I think starting with a concrete image definitely helps. I had a hard time with learning when to use perlin noise or random() function calls when coloring tiles. I am very happy with the result and I believe it does look like an infinite rainbow road, landscapes, atoms, and patterns.
